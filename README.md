# Excel Data Analyzer

Projeto desenvolvido em Python para análise automática de planilhas de vendas.

O sistema lê um arquivo Excel contendo dados de produtos e retorna informações relevantes como faturamento, lucro, margem média e produtos com maior volume de vendas.

## Tecnologias utilizadas

- Python
- Pandas
- OpenPyXL

## Estrutura da planilha

A planilha analisada deve conter as seguintes colunas:

- Produto
- Marca
- Valor de Venda
- Custo
- Região de Venda

## Métricas analisadas

O programa gera automaticamente:

- Total de registros
- Faturamento total
- Custo total
- Lucro total
- Margem média
- Produto mais vendido
- Marca mais vendida
- Região com maior faturamento
